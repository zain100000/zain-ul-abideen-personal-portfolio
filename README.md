# Muhammad Zain-ul-Abideen | Portfolio

A modern anime-inspired developer portfolio built with Next.js, Tailwind CSS, and Framer Motion. Includes a Dragon Ball Z themed skills section, animated page transitions, and a working contact form.

## Overview

- Theme: Anime Style
- UI: Dark mode with purple and pink accents
- Animations: Framer Motion transitions and scroll effects
- Contact: EmailJS-powered contact form

## Pages

- **Home**: Hero section with animated introduction
- **About**: Professional summary with animated stats
- **Skills**: Dragon Ball themed skill showcase
- **Projects**: Featured project cards with character accents
- **Experience**: Timeline layout with avatars
- **Education**: Visual education timeline
- **Contact**: EmailJS contact form

## Tech Stack

| Technology    | Purpose                         |
| ------------- | ------------------------------- |
| Next.js 14    | App Router and server rendering |
| React         | UI components and hooks         |
| Tailwind CSS  | Styling and responsive layout   |
| Framer Motion | Animations and transitions      |
| EmailJS       | Contact form email delivery     |
| React Icons   | Icon library                    |
